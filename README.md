# Projeto Fullstack (Angular + Spring Boot)

Monorepo de estudo com **frontend** (Angular) e **backend** (Spring Boot) desacoplados, com autenticação **JWT + refresh token**, CRUD e permissões **RBAC**.

## Estrutura
- `frontend/` — aplicação Angular
- `backend/` — API Spring Boot
- `docker/` — infraestrutura local (Postgres etc.)
- `docs/` — documentação de arquitetura/decisões
- `scripts/` — scripts auxiliares
